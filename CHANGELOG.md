# Changelog

## 0.2.0 — Theme, Thank You Redirect, and Admin CMS

- Added a robust dark/light theme toggle with readable text in both modes.
- Added early theme initialization to avoid incorrect theme flashes.
- Added a thank-you page after form submission.
- Added automatic redirect from the thank-you page back to the homepage.
- Added a protected admin CMS API.
- Added CMS storage layer for posts, pages, and site information.
- Added a full admin dashboard with tabs for overview, messages, tests, posts, pages, and site info.
- Added public `/blog` page for published posts.
- Added public `/blog/[id]` page for single posts.
- Added CMS documentation.
- Updated deployment documentation for FormSubmit, Vercel KV, and admin CMS.
- Removed unused Resend dependency.
