# AK-AD media — Project Brief

## Overview

AK-AD media is a professional digital marketing and social media management website targeting small and medium Syrian businesses.

The website should communicate that AK-AD media is not only a design provider, but a practical growth partner that understands the Syrian market and helps businesses build trust, visibility, and customer acquisition online.

## Technical Notes

- Use Next.js for Vercel deployment.
- Keep Arabic as the default language.
- Store secrets only in environment variables.
- Use Vercel KV for persistent lead/test result storage when configured.
- Avoid placing customer data in public files.
